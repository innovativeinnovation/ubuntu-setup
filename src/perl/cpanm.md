# cpanminus

[cpanminus](https://metacpan.org/pod/App::cpanminus) is a script to get,
unpack, build and install modules from CPAN.

## Installation

To install:

```bash
sudo apt install -y cpanminus
```

## Usage

To check cpanm version:

```bash
cpanm -V
```

To install a package:

```bash
cpanm <package>
```
