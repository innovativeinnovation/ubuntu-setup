# RubyGems

[RubyGems](https://github.com/rubygems/rubygems) is a package management
framework for Ruby.

## Installation

RubyGems is likely already installed in your Ruby environment, you can check
by running:

```bash
gem --version
```

##Â Usage

To install a gem:

```bash
gem install <gem>
```

To update installed gems:

```bash
gem update [<gem>]
```

To output installed gems:

```bash
gem list
```
